# **Evidence pojištěnců BUNNY**
Evidence pojištěnců BUNNY je projekt vytvořen k závěrečné zkoušce rekvalifikačního kurzu společnosti [_ITNetwork.cz_](https://www.itnetwork.cz/). 

Jedná se o jednoduchou konzolovou aplikaci, která umí vkládat, hledat a vypisovat vložené hodnoty - zde tedy osoby/pojištěnce.

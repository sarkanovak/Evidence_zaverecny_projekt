# **Evidence pojištěných BUNNY**
Evidence pojištěných BUNNY je projekt vytvořen k závěrečné zkoušce rekvalifikačního kurzu společnosti [_ITNetwork.cz_](https://www.itnetwork.cz/). 

Jedná se o jednoduchou konzolovou aplikaci, která umí vkládat, hledat, vypisovat vložené hodnoty - zde tedy osoby/pojištěnce a ukončit sama sebe. Má i pár přidaných hodnot:
- Při vkládání nového záznamu má podmínky, aby jméno či věk bylo odpovídajícího datového typu (tedy aby v jméně nešly ukládat číslice apod.)
    - pomocí _isdigit(), isaplha()_
- Déle pokud při vkládání záznamu napíšete obě části jména malými písmeny, ve výpisu text automaticky převede na první velké počáteční písmeno
    - pomocí _.title()_
- Milé překvapení po ukončení aplikace :)
